# JavaSTG 团队主页

## 团队介绍

欢迎来到 JavaSTG 团队！我们是一个专注于维护和发展 Java 射击游戏（STG）项目的开源团队。成立于 2026 年，我们致力于通过协作开发，打造高质量、可扩展的 STG 游戏框架和示例。

## 项目概述

JavaSTG 是一个基于 Java 开发的射击游戏框架，提供了以下核心功能：

- 游戏对象系统（玩家、敌人、子弹、道具等）
- 碰撞检测系统
- 游戏状态管理
- 输入处理
- 资源管理
- 关卡系统

项目结构清晰，代码组织合理，便于扩展和定制。

## 贡献指南

我们欢迎所有对 STG 游戏开发感兴趣的开发者加入我们的团队！以下是贡献指南：

1. **代码贡献**
   - 遵循项目的代码风格（参考 `doc/CODE_STYLE.md`）
   - 提交前确保代码能够正常编译和运行
   - 提供清晰的 commit 信息

2. **文档贡献**
   - 帮助完善项目文档
   - 提供使用示例和教程

3. **问题反馈**
   - 在 GitHub Issues 中报告 bug
   - 提出新功能建议

## 开发流程

1. 从主分支（main）创建新的功能分支
2. 在分支上进行开发
3. 提交代码并创建 Pull Request
4. 团队成员进行代码审查
5. 合并到主分支

## 团队目标

1. 完善和扩展 JavaSTG 框架功能
2. 提供更多游戏示例和关卡
3. 建立活跃的开发者社区
4. 推动 Java 在游戏开发领域的应用

## 愿景

我们希望 JavaSTG 能够成为一个成熟、稳定的 STG 游戏开发框架，为开发者提供便捷的工具和资源，同时也能为玩家带来优质的游戏体验。
